# dark-side-chat
A chat that leads to the dark side

## Used Frameworks
- https://www.meteor.com
- https://angularjs.org/
- https://material.angularjs.org/latest/#/

## Setup
- Install Meteor (https://www.meteor.com/install)
- cd into the project directory and type *meteor* to start the server
